Wie sich die Website-Geschwindigkeit auf das Geschäftsergebnis auswirkt

Es gibt viele Studien, die bestätigen, was viele Unternehmen und Webentwickler bereits wissen: Geschwindigkeit beeinflusst den Umsatz. Bessere Web-Performance bedeutet höhere Nutzerbindung und mehr Konversionen. Eine bessere Performance führt auch zu besserer SEO, was die gesamten Zugriffszahlen erhöht und die Lead-Generierung, den Umsatz oder die Werbeeinnahmen steigert.

Eine Studie von Google und Bing kam zu dem Schluss, dass eine Verzögerung von zwei Sekunden beim Seitenaufbau einer Website durchschnittlich zu einem Umsatzverlust von 4,3 % pro Besucher führte. Eine weitere unabhängige Studie hat gezeigt, dass eine zusätzliche Sekunde Ladezeit die Konversionen um 7 % reduziert.

Wie wirkt sich die Verwendung von Cloudflare auf die Performance aus?

Tests von Cedexis (einem Unternehmen, das die CDN-Performance bewertet) haben Folgendes ergeben:

  • Cloudflare ist für das 95. Perzentil über HTTPS mehr als 50 Millisekunden oder 23 % schneller als der nächste Wettbewerber
  • Cloudflare schneidet über HTTPS sowohl beim 50. als auch beim 95. Perzentil besser ab als alle Wettbewerber
  • Cloudflare schneidet über HTTP sowohl beim 50. als auch beim 95. Perzentil besser ab als alle Wettbewerber

Die schnelle und autoritative Übersetzung von Domainnamen in IP-Adressen erfordert einen unternehmenstauglichen DNS.  Jede Webseite benötigt zum Laden mehrere Anfragen und Antworten, sodass sich bei der Beantwortung von DNS-Abfragen eine Verbesserung um einige Millisekunden schnell summiert.  Deshalb ist es uns so wichtig, dass unsere DNS Resolver die schnellsten sind, die es gibt.   • Cloudflare ist der schnellste öffentliche Resolver, weltweit fast 30 % schneller als der nächstschnellste öffentliche DNS Resolver
