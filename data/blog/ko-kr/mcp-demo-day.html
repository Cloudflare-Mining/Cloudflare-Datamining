10분 읽기
이 게시물은 English, 繁體中文, 日本語, Português, Español 및 简体中文로도 이용할 수 있습니다.

오늘, Cloudflare에서는 Anthropic, Asana, Atlassian, Block, Intercom, Linear, PayPal, Sentry, Stripe, Webflow와 협업하여 완전히 새로운 원격 MCP 서버 세트를 제공하게 되어 기쁩니다. 이 서버 세트는 모두 Cloudflare를 기반으로 구축되어 Claude 사용자가 채팅 인터페이스를 벗어나지 않고도 프로젝트 관리, 청구서 생성, 데이터베이스 쿼리, 전체 스택 애플리케이션 배포까지 모두 수행할 수 있도록 해줍니다. 

11월에 Anthropic에서 모델 컨텍스트 프로토콜(MCP)을 소개한 이후, 이에 대한 관심이 점점 더 커지면서 거의 매일 새로운 MCP 서버가 출시되는 것 같습니다. 그럴 만한 이유가 있습니다!  MCP는 AI 에이전트를 현실로 만드는 데 누락된 조각이었으며, AI 에이전트가 도구와 상호 작용하du 조치를 취하고 추가 컨텍스트를 확보하는 방식을 정의하는 데 도움이 되었습니다.

하지만 지금까지는 최종 사용자가 MCP 서버를 사용하려면 로컬 컴퓨터에 MCP 서버를 설치해야 했습니다. 오늘 Anthropic에서 통합을 발표했으므로 웹 사이트에서와 동일한 방식으로 MCP 서버에 액세스할 수 있습니다. URL을 입력하기만 하면 됩니다.

Cloudflare에서는 원격 MCP 서버의 개발을 간소화하는 도구를 구축하는 데 집중해왔으며, 덕분에 고객의 엔지니어링 팀에서는 프로토콜의 복잡성을 관리하는 대신 애플리케이션을 위한 MCP 도구를 구축하는 데 시간을 집중할 수 있습니다. Cloudflare에서 원격 MCP 서버를 배포하는 것이 얼마나 쉬운지 궁금하신 분에게는 클릭 한 번으로 최신 MCP 표준을 지원하도록 미리 구축된 MCP 서버를 얻을 수 있다는 사실을 기꺼이 알려드립니다. 

하지만 저희의 말을 그대로 받아들일 필요는 없습니다. 직접 확인해 보세요! 업계 리더들은 사용 편의성을 활용하여 Cloudflare에서 MCP 서버를 구축하여 사용자에게 새로운 AI 기반 경험을 제공하고 있으며, 이제 여러분도 'Cloudflare에 배포'를 클릭하여 동일한 작업을 시작할 수 있습니다. 

계속 읽으면서 이들 기업에서 사용자를 위해 제공하는 새로운 기능과 이를 제공할 수 있었던 방법에 대해 자세히 알아보세요. 아니면 5월 1일(오늘) 오전 10시(미국 태평양 표준시)에 열리는 데모 데이에 참여하여 이 제품의 작동 방식을 확인해 보세요. 

저희는 또한 Cloudflare의 원격 MCP 서버를 현재 고객에게 제공하고 있으며 이를 구축하면서 얻은 교훈을 공유합니다. 

MCP: 차세대 애플리케이션 지원

매장, 부동산 중개인, 서비스 등 모든 서비스에 웹 사이트가 있을 것으로 항상 기대되었던 것은 아닙니다. 그러나 더 많은 사람들이 인터넷 연결에 액세스할 수 있게 되면서 이는 빠르게 현실이 되었습니다.

현재 모든 웹 사용자가 AI 도구에 액세스하여 다양한 작업을 수행하는 비슷한 전환기에 있습니다. 개발자라면 코드를 작성할 때 가장 먼저 Claude와 같은 도구를 사용하는 경우가 많습니다. 그런 경우 Claude가 코드를 작성하는 데 도움이 되었다면 배포하는 데도 도움이 될 것입니다.

또는 개발자가 아니더라도 Claude가 레시피를 생각해내는 데 도움이 되었다면 필요한 식료품을 주문하는 데도 도움이 될 것입니다. 

원격 MCP를 사용하면 이제 이러한 모든 시나리오가 가능합니다. 그리고 웹에 최초로 구축한 기업들이 퍼스트 무버 우위를 차지했던 것처럼, MCP 포워드 방식으로 먼저 구축된 기업들이 이점을 얻을 가능성이 높습니다. 

사용자가 제품의 귀사의 가치를 더 빨리 경험할수록 귀사의 제품을 이어받고 업그레이드하며 계속 사용할 가능성이 높아집니다. MCP를 통해 서비스를 에이전트에 연결하면 사용자는 필요한 것을 요청하기만 하면 에이전트가 나머지를 처리하므로 사용자는 그 "아하" 하는 순간을 더 빨리 맞이할 수 있습니다. 

MCP로 AI 우선 서비스 만들기

MCP를 채택한 기업에서는 다음과 같은 효과를 빠르게 확인할 수 있습니다. 

진입 장벽 낮추기

모든 사용자가 대시보드를 배우거나 문서를 자세히 읽어 제품을 이해할 시간이 있는 것은 아닙니다. MCP를 사용하면 그럴 필요가 없습니다. 사용자가 원하는 내용만 설명하면 나머지는 에이전트가 알아서 처리합니다. 

맞춤형 경험 만들기

MCP는 사용자의 요청과 상호 작용을 추적할 수 있으므로 향후 도구 호출을 사용자의 사용 패턴과 선호도에 맞게 조정할 수 있습니다. 이를 통해 각자가 실제로 제품을 사용하는 방식에 따라 보다 개인화되고 관련성 있는 경험을 쉽게 제공할 수 있습니다.

자연스럽게 업그레이드 유도

AI 에이전트는 기능 비교 표에 의존하는 대신, 상위 등급 요금제를 사용하면 특정 사용자가 자신의 목표를 달성하는 데 어떻게 도움이 되는지 설명할 수 있습니다. 

새로운 기능 및 통합 제공

모든 통합이나 경험을 자체적으로 구축할 필요는 없습니다. MCP를 통해 도구를 노출하면 사용자가 제품을 나머지 스택에 연결할 수 있습니다. 에이전트가 공급자 전체에 걸쳐 도구를 결합할 수 있으므로 모든 타사 서비스를 직접 지원할 필요 없이 통합이 가능합니다.

왜 Cloudflare에 MCP를 구축해야 할까요? 

MCP 출시 이후, 저희는 개발자들이 원격 MCP 서버를 빌드하고 배포하는 것을 어떻게 쉽게 만들었는지에 대한 내용을 공유해 왔습니다. 여기에는 프로토콜 복잡성을 추상화하여 인증 및 전송을 처리하는 것부터 완전히 상태 유지형 MCP 서버를 지원하는 것까지(기본적으로) 유휴 비용을 최소화하기 위해 사용되지 않을 때는 '절전 모드'로 전환하는 것까지 포함됩니다.

Cloudflare에서는 MCP 표준의 최신 변경 사항에 대한 지원을 추가하고 서버를 프로덕션으로 전환하는 것을 훨씬 더 쉽게 하기 위해 정기적으로 업데이트를 계속 제공하고 있습니다.

  • 새로운 Streamable HTTP 전송 지원: 이제 MCP 서버에서 SSE와 함께 최신 Streamable HTTP 전송을 사용할 수 있으므로 최신 표준과의 호환성이 보장됩니다. 저희는 AI Playground와 mcp-remote 프록시도 업데이트하여 새로운 전송을 쉽게 테스트할 수 있는 원격 MCP 클라이언트를 제공합니다.

  • MCP 서버에 대한 Python 지원: 이제 JavaScript/TypeScript 뿐만 아니라 Python을 사용하여 Cloudflare에 MCP 서버를 구축할 수 있습니다. 

  • 문서, 스타터 템플릿, 배포 흐름 개선: Cloudflare에서는 새로운 빠른 시작 템플릿을 추가하고, 새로운 전송 방법을 다루기 위해 문서를 확장했으며, 학습한 내용을 바탕으로 MCP 서버 구축의 모범 사례를 공유했습니다. 

하지만 직접 설명하기보다는 고객이 어떤 제품을 구축했는지, MCP 배포를 위해 왜 Cloudflare를 선택했는지 직접 보여드리는 것이 좋겠다고 생각했습니다. 

현재 연결할 수 있는 원격 MCP 서버

Asana

오늘날 업무는 다양한 애플리케이션과 서비스를 이용하여 수행됩니다. Asana에서는 MCP에 투자함으로써 현 위치에 있는 사용자를 만나고 도구와 워크플로우 전반에 걸쳐 에이전트 기반 상호 운용성을 실현할 수 있습니다. 사용자는 자연어를 사용하여 Asana Work Graph와 상호 작용해서 MCP 클라이언트로부터 프로젝트 업데이트를 받고, 작업을 검색하며, 프로젝트를 관리하고, 의견을 보내고, 기한을 업데이트할 수 있습니다.

사용자는 다양한 작업을 원활하게 조율하고 통합할 수 있습니다. 그 예로는 다른 애플리케이션의 프로젝트 계획 또는 회의 메모를 Asana에서 완전히 할당된 작업 세트로 전환하기, 구현을 위해 Asana 작업을 MCP 지원 IDE로 직접 가져오기 등이 있습니다. 이러한 유연성 덕분에 시스템 전반의 작업을 그 어느 때보다 더 쉽고 빠르며 자연스럽게 관리할 수 있습니다.

자사 MCP 서버의 출시를 가속화하기 위해 Asana에서는 Cloudflare의 도구를 기반으로 구축했으며, 기본 인프라를 활용하여 빠르게 진화하는 이 분야에서 신속하고 안정적으로 대응할 수 있었습니다.

"Asana에서는 항상 모든 팀이 업무를 쉽게 조정할 수 있도록 지원하는 데 집중해 왔습니다. MCP는 Work Graph를 Claude.ai와 같은 AI 도구에 직접 연결하여 AI가 업무 관리의 진정한 동료가 될 수 있도록 합니다. 저희는 통합을 통해 자연어를 구조화된 작업으로 변환하여 회의 메모에서 프로젝트를 생성하거나 업데이트를 AI로 가져옵니다. Cloudflare의 인프라를 기반으로 구축하여 인증 및 확장을 처리해서 신속하게 배포할 수 있었으며, 사용자에게 최고의 경험을 제공하는 데 집중했습니다." – Prashant Pandey, Asana CTO

여기에서 Asana의 MCP 서버에 대해 자세히 알아보세요. 

Atlassian

Jira 및 Confluence 클라우드 고객은 이제 베타 버전으로 제공되는 Atlassian 원격 MCP 서버를 통해 Anthropic의 Claude 애플리케이션에서 데이터와 안전하게 직접 상호 작용할 수 있습니다. Cloudflare 인프라에서 호스팅되는 사용자는 허가된 범위 내에서 데이터를 안전하게 유지하면서 작업을 요약하고, 이슈 또는 페이지를 생성하며, 다단계 작업을 수행할 수 있습니다.

사용자는 어디에서든 Claude를 사용하여 Jira 및 Confluence의 정보에 액세스하여 다음을 수행할 수 있습니다.

  • Jira 작업 항목 또는 Confluence 페이지 요약

  • Claude에서 바로 Jira 작업 항목 또는 Confluence 페이지 생성

  • 모델이 이슈 또는 대량 페이지 생성과 같은 여러 작업을 한 번에 수행하도록 함

  • Claude가 액세스할 수 있는 다양한 소스의 컨텍스트로 Jira 작업 항목 강화

  • 그 밖에도 많은 기능이 있습니다!

“AI는 만능 솔루션이 아니며, 팀의 업무에 AI가 내장되어야 한다고 생각합니다. 그렇기 때문에 MCP에 투자하고 MCP가 이미 업무에 투입되고 있는 더 많은 곳에서 여러 팀을 만날 수 있게 되어 매우 기쁩니다. Cloudflare 인프라에서 호스팅한다는 것은 이러한 강력한 통합을 고객에게 더 빠르게 제공하고 Jira와 Confluence를 사용하여 그 어느 때보다 많은 작업을 수행할 수 있도록 지원하는 동시에 기업 데이터를 안전하게 보호할 수 있음을 의미합니다. Cloudflare에서 OAuth부터 즉시 사용 가능한 원격 MCP 지원까지 모든 것을 제공했기 때문에 우리는 완벽하게 운영되는 설정을 신속하게 구축하고 보호하며 확장할 수 있었습니다.” — Taroon Mandhana, Atlassian 제품 엔지니어링 책임자

Atlassian의 MCP 서버에 대해 여기에서 자세히 알아보세요. 

Intercom

Intercom에서는 AI의 변혁적인 힘이 점점 더 드러나고 있습니다. Intercom의 AI 에이전트인 Fin은 이제 Anthropic과 같은 선도적인 기업의 고객 지원 대화 중 50% 이상을 자율적으로 해결하고 있습니다. MCP를 사용하면 AI를 내부 도구 및 시스템에 그 어느 때보다 쉽게 연결하여 비즈니스 가치를 높일 수 있습니다.

예를 들어, 고객과의 대화에서 제품이 어떻게 사용되는지, 고객이 경험하는지에 대한 귀중한 인사이트가 제공됩니다. 그러나 이 데이터는 지원 플랫폼 내에 잠겨 있는 경우가 많습니다. Intercom MCP 서버는 이처럼 풍부한 고객 데이터 소스를 활용하여 AI 도구를 사용하는 조직 내의 모든 사람이 액세스할 수 있도록 합니다. 예를 들어, 엔지니어는 Cursor 또는 Claude Code와 같은 도구에서 Intercom의 대화 기록과 사용자 데이터를 활용하여 문제를 보다 효율적으로 진단하고 해결할 수 있습니다.

“MCP의 추진력은 흥미롭습니다. MCP 덕분에 Claude.ai와 같은 어시스턴트와 Fin과 같은 에이전트를 시스템에 점점 더 쉽게 연결하고 실제 업무를 수행할 수 있습니다. Cloudflare의 툴킷은 이러한 움직임을 더욱 가속화하고 있습니다. Intercom의 명확한 문서, 목적에 맞게 구축된 도구, 개발자 우선 플랫폼 덕분에 Intercom에서는 하루 만에 개념부터 프로덕션까지 완료할 수 있었고, 이를 통해 Intercom MCP 서버를 쉽게 출시할 수 있었습니다. 저희는 고객들이 Cloudflare를 활용하여 자체 MCP 서버를 구축하고 배포하여 내부 시스템을 Fin 및 다른 클라이언트에 안전하고 안정적으로 연결할 수 있도록 권장할 계획입니다.” — Jordan Neill, Intercom 엔지니어링 수석 부사장

Linear

Linear MCP 서버를 통해 사용자는 필요할 때 문제 및 제품 개발 프로세스의 컨텍스트를 AI 어스스턴트에 직접 가져올 수 있습니다. Claude에서 제품 사양을 개선하든, Cursor와 버그를 수정하기 위해 협업하든, 이메일에서 즉석에서 문제를 생성하든 상관없습니다. 

“저희는 Cloudflare를 기반으로 빠르게 변화하는 공간에서 Cloudflare의 프레임워크를 활용하고 에지에서 유연하고 빠른 컴퓨팅을 제공합니다. 저희는 MCP를 통해 Linear의 문제 추적 및 제품 개발 워크플로우를 선택한 AI 도구에 직접 통합하므로 팀에서 컨텍스트를 전환할 필요가 없습니다. 저희 목표는 간단합니다. 개발자와 제품팀이 Claude에서 사양을 개선하든, Cursor에서 디버깅을 하든, 대화를 통해 문제를 만들든 이미 있는 곳에서 작업에 액세스할 수 있도록 하는 것입니다. 이러한 원활한 통합을 통해 고객은 흐름을 유지하고 훌륭한 제품을 만드는 데 집중할 수 있습니다” — Tom Moor, Linear 미국 엔지니어링 책임자

Sentry

Sentry MCP 서버를 사용하면 개발자는 IDE, Claude 등의 어시스턴트에서 바로 Sentry의 컨텍스트를 쿼리하여 프로젝트 전체 또는 개별 파일에 대한 오류 및 문제 정보를 가져올 수 있습니다.

개발자는 또한 MCP를 사용하여 프로젝트를 생성하고, 설정 정보를 캡처하며, 프로젝트 및 조직 정보를 쿼리하고, 이 정보를 사용하여 Sentry용 애플리케이션을 설정할 수도 있습니다. 저희는 MCP를 지속해서 구축해 나가면서, 팀에서 저희 에이전트인 Seer로부터 근본 원인 분석 및 솔루션 정보를 가져올 수 있도록 하고, 추적, 예외 처리 등의 Sentry 기능을 위한 계측을 간소화할 수도 있습니다. 

저희는 Cloudflare에서 이를 호스팅하고 원격 MCP를 사용하여 확장이나 인증 등 로컬에서 실행하려고 할 때 발생하는 여러 가지 복잡한 문제를 피할 수 있었습니다. 원격 MCP를 사용하면 Sentry의 자체 OAuth 구성을 직접 활용할 수 있습니다. 또한 Durable Object 지원을 통해 MCP 내에서 상태를 유지할 수 있으며, 이는 로컬로 실행하지 않을 때 중요합니다. 

“Sentry의 약속은 항상 개발자에게 있어 프로덕션 소프트웨어를 더 쉽게 안정적으로 실행할 수 있도록 하겠다는 것이며, AI 시대에는 더욱 그런 방향으로 추진할 계획입니다. 개발자들은 MCP와 같은 도구를 활용하여 스택을 AI 모델 및 데이터 소스와 통합하고 있습니다. 저희가 Cloudflare에서 MCP를 구축하기로 선택한 이유는 개발자가 소프트웨어를 더 쉽게 출시한다는 비전을 공유하고, 팀에서 차세대 AI 에이전트를 구축하고 안전하게 실행할 수 있도록 하는 데 모두 투자했기 때문입니다. 이러한 통합으로 인해 발생하는 복잡한 상호 작용을 디버깅하는 것이 점점 더 중요해지고 있으며, Sentry에서는 문제를 신속하게 진단하고 해결하는 데 필요한 필수 가시성을 제공합니다. MCP는 이 중요한 Sentry 컨텍스트를 개발자 워크플로우에 직접 통합하여 팀에서 안정적인 애플리케이션을 일관되게 구축하고 배포할 수 있도록 지원합니다.” — David Cramer, Sentry CPO 겸 공동 설립자

Block 

Square의 API는 판매자가 결제를 받고, 주문을 생성하고 추적하며, 재고를 관리하고, 고객을 정리하는 등의 작업을 지원하는 포괄적인 도구 세트입니다. 이제 전용 Square MCP 서버를 통해 판매자는 AI 에이전트의 도움을 받아 Square의 전체 API 리소스 및 엔드포인트 제품군을 활용해 비즈니스를 구축할 수 있습니다. Claude 및 codename goose와 같은 AI 에이전트와 통합함으로써, 판매자는 Square의 기능을 최대한 활용하는 정교하고 맞춤화된 사용 사례를 보다 낮은 기술적 장벽으로 구현할 수 있습니다.

Webflow

Webflow MCP 서버는 CMS 관리, SEO 감사 및 개선, 콘텐츠 현지화, 사이트 게시 등을 지원합니다. 이를 통해 사용자는 AI 에이전트에서 직접 사이트를 관리하고 개선할 수 있습니다. 

"저희는 MCP가 Webflow와 상호 작용하는 새로운 방법으로, 모든 사람에게 강력한 개발 기능을 제공한다는 저희 사명에 잘 부합한다고 생각합니다. 저희는 MCP를 단순히 API의 다른 표면이라고 생각하지 않으며, 웹 사이트 게시, CMS 컬렉션 생성, 콘텐츠 업데이트 등 MCP가 지원하는 작업의 관점에서 생각합니다. MCP를 사용하면 이러한 작업을 검색 가능하고 안전하며 컨텍스트에 기반한 방식으로 노출할 수 있습니다. MCP 덕분에 AI와 인간이 솔루션을 짜맞추거나 저수준 API 세부 정보를 처리할 필요 없이 나란히 작업할 수 있는 새로운 워크플로우가 가능해집니다. Cloudflare에서는 최신 웹 인프라를 구축하는 데 필요한 안정성, 성능, 개발자 도구를 제공하여 이러한 목표를 지원합니다. 원격 MCP 서버에 대한 Cloudflare의 지원은 성숙하고 잘 통합되어 있으며, 인증 및 내구성에 대한 접근 방식은 Cloudflare에서 이러한 서비스의 규모와 보안에 대해 생각하는 방식과 일치합니다.” — Utkarsh Sengar, Webflow 엔지니어링 부사장
