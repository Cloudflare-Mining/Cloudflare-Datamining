El código abierto es el tejido central de la web, y las herramientas de código abierto que impulsan la web moderna dependen de la estabilidad y el soporte de la comunidad. 

Para garantizar que dos importantes proyectos de código abierto tengan los recursos que necesitan, nos enorgullece anunciar nuestro patrocinio financiero a dos marcos fundamentales en el ecosistema web moderno: Astro y TanStack.

Creemos que es importante que no hagamos esto solos. Para que la web abierta siga prosperando, debemos apostar por tecnologías y marcos que sean abiertos y accesibles para todos, y que no dependan de ninguna empresa. 

Por eso, también nos complace anunciar que para estos patrocinios estamos uniendo fuerzas con nuestros colegas de Netlify para patrocinar TanStack y Webflow para patrocinar Astro.

¿Por qué elegir Astro y TanStack? Invertir en el futuro del frontend

Nuestra decisión de apoyar a Astro y TanStack fue deliberada. Estos dos proyectos representan visiones distintas pero complementarias para el futuro del desarrollo web. Uno de ellos redefine la arquitectura de los sitios web de alto rendimiento basados en el contenido, mientras que el otro ofrece un conjunto de herramientas integrales para crear las aplicaciones web más ambiciosas.

Astro: el marco para los sitios de alto rendimiento 

Cuando se trata de respaldar una tecnología, creemos que las acciones valen más que las palabras. 

Por eso, nuestro apoyo a Astro no es solo financiero, sino fundamental. Administramos nuestro sitio de documentación para desarrolladores, developers.cloudflare.com, completamente en Astro. Este no es un proyecto paralelo pequeño, es un recurso fundamental que visitan cientos de miles de desarrolladores todos los días, y docenas de colaboradores lo mantienen actualizado de forma constante. Para un sitio como este, el rendimiento no es una característica; es un requisito. 

Elegimos Astro porque sus principios básicos son similares a los nuestros. Su arquitectura "cero JavaScript por defecto" ofrece el rendimiento bruto y la optimización de motores de búsqueda (SEO) estelar que exige un sitio con mucho contenido, lo que garantiza que nuestros documentos sean rápidos y reconocibles. Y lo que es igual de importante, Astro es independiente del marco, lo que permite a los equipos utilizar componentes de React, Vue o Svelte sin depender de un proveedor. 

Astro facilita a nuestro equipo global la actualización del contenido y, lo que es más importante, la rapidez de la documentación. Nuestro patrocinio es el resultado directo del inmenso valor que hemos experimentado de primera mano.   

La asociación y el soporte de Cloudflare reafirman nuestra misión compartida: hacer que la web sea más rápida, más abierta y mejor para todos los que crean en ella.  - Fred K. Schott, cocreador de Astro, administrador del proyecto

Webflow ofrece a los especialistas en marketing, diseñadores y desarrolladores la libertad de crear sin riesgos. Astro comparte ese mismo espíritu al eliminar barreras, acelerar los flujos de trabajo y abrir nuevas posibilidades creativas. Junto con Cloudflare y Netlify, estamos ayudando a garantizar que las herramientas de las que depende nuestra comunidad permanezcan abiertas, sostenibles y preparadas para el futuro. - Allan Leinwand, director técnico de Webflow

TanStack Start: el marco integral para aplicaciones ambiciosas

Si Astro proporciona la base ideal para sitios con mucho contenido, TanStack ofrece el motor ideal para aplicaciones web complejas. TanStack no es un marco único, sino un conjunto de bibliotecas potentes, sin interfaz gráfica y con seguridad de tipos que resuelven los problemas más difíciles en el desarrollo de aplicaciones modernas.

Las bibliotecas como TanStack Query se han convertido en el estándar del sector para gestionar el estado asincrónico del servidor, y resuelve con elegancia desafíos complejos como el almacenamiento en caché, la recuperación en segundo plano y las actualizaciones optimistas que antes requerían miles de líneas de código frágil y personalizado. Del mismo modo, TanStack Router ofrece seguridad completa al enrutamiento, y elimina toda una clase de errores comunes, mientras que TanStack Table y TanStack Form proporcionan las primitivas sólidas y sin interfaz gráfica necesarias para crear interfaces de usuario sofisticadas y con un uso intensivo de datos.

Y hoy, TanStack anunció el lanzamiento oficial de la versión candidata para TanStack Start 1.0, dando un gran paso hacia la preparación para la producción.

TanStack Start es un nuevo marco integral que ofrece estas potentes bibliotecas en una experiencia de desarrollo consistente y de nivel empresarial. Con funciones como la representación del lado del servidor (SSR) de documentos completos, la transmisión y una arquitectura de "implementación en cualquier lugar", TanStack Start está diseñado para el perímetro moderno sin servidor. Brinda la potencia y la seguridad necesarias para aplicaciones ambiciosas y es la combinación perfecta para entornos de implementación como Cloudflare Workers.

Con Cloudflare junto a nosotros, TanStack puede seguir elevando el nivel de herramientas rápidas, escalables y de tipos seguros para impulsar la próxima generación de aplicaciones web y, al mismo tiempo, proteger la apertura y la libertad de las que dependen los desarrolladores. - Tanner Linsley, creador de TanStack

Apoyar una web abierta no es algo que nos guste, sino un requisito para que podamos cumplir nuestra misión de crear una web mejor. Colaborar con Cloudflare para garantizar que estos importantes proyectos reciban financiación es la decisión más fácil que podemos tomar. - Mat B, director general
