<h3><strong>Job Summary</strong></h3>
<p>The <strong>Senior Integration Developer - Oracle Cloud Integrations</strong> is the definitive technical expert responsible for developing, maintaining, governing, and standardizing all data movement involving our <strong>Oracle Fusion ERP/EPM Cloud</strong> environment. This highly influential role acts as the primary liaison between the Finance &amp; Legal IT team and the central <strong>Enterprise Architecture (EA) Organization</strong>, ensuring that all Oracle integrations—primarily those built on <strong>Oracle Integration Cloud (OIC)</strong>—adhere to current Enterprise Reference Architecture, security standards, and integration best practices.</p>
<p>This role will design the most complex and critical integration patterns for the entire Oracle footprint (Finance, SCM, EPM).</p>
<h3><strong>Key Responsibilities</strong></h3>
<h4><strong>Enterprise Integration Architecture &amp; Governance</strong></h4>
<ul>
	<li><strong>Reference Architecture Enforcement:</strong> Act as the gatekeeper for all Oracle integration design, ensuring strict adherence to the standards, patterns, and protocols defined by the larger Enterprise Architecture organization (e.g., API-led connectivity, asynchronous messaging patterns, canonical data models).</li>
	<li><strong>Technical Roadmap:</strong> Define the long-term technical roadmap for the <strong>Oracle Integration Cloud (OIC)</strong> platform itself, ensuring the environment is scaled, secured, and leverages the latest features in alignment with Enterprise Architecture strategy.</li>
	<li><strong>Design Review:</strong> Lead formal technical design reviews for all high-risk or high-volume integrations (e.g., data flows for OTC, PTP, and SCM execution) to validate technical soundness, performance, and compliance with security and audit requirements.</li>
	<li><strong>Tooling Best Practices:</strong> Establish best practices for using OIC components (e.g., orchestrations, adapters, security) across all development teams, including internal staff and System Integrator partners.</li>
</ul>
<h4><strong>Solution Design &amp; Technical Leadership</strong></h4>
<ul>
	<li><strong>Complex Integration Design:</strong> Design the most challenging, cross-functional, and real-time integration architectures for the Oracle Fusion program, such as complex master data synchronization (e.g., EDM to Fusion), multi-cloud data exchanges, and high-volume transaction routing.</li>
	<li><strong>Technical Mentorship:</strong> Serve as the top integration expert, providing technical leadership, coaching, and direction to the Sr. Technical Lead, BSAs, and external development teams building integrations.</li>
	<li><strong>Security &amp; Performance:</strong> Define and validate the technical security protocols for all endpoints and interfaces (e.g., OAuth, API Gateway configuration), ensuring data transmission is compliant and performant.</li>
</ul>
<h4><strong>Operational &amp; ALM Support</strong></h4>
<ul>
	<li><strong>OIC Administration Liaison:</strong> Partner with the ALM team (Senior Technical Lead and Analyst) to define OIC environment management standards, deployment automation, and monitoring strategy, specifically focusing on integration health and error handling.</li>
	<li><strong>Incident Resolution:</strong> Provide expert-level assistance in troubleshooting and root cause analysis for persistent, complex integration failures that impact business continuity.</li>
	<li><strong>Documentation:</strong> Create and maintain high-quality architectural documentation, integration patterns, and service catalogs for all Oracle interfaces.</li>
</ul>
<h3><strong>Required Qualifications</strong></h3>
<ul>
	<li><strong>10+ years</strong> of previous experience in IT, with a focus on Enterprise Architecture, Integration, or Middleware Platforms.</li>
	<li><strong>6+ years</strong> of direct experience architecting, designing, and governing enterprise-level integrations for major ERP/SCM/EPM/GTM systems (SAP, Oracle EBS, or Oracle Fusion).</li>
	<li><strong>Expert-level, hands-on knowledge of Oracle Integration Cloud (OIC)</strong>, including strong familiarity with its various adapters (REST, SOAP, ERP, Database), mapping, and orchestration capabilities.</li>
	<li>Proven experience working directly with a central <strong>Enterprise Architecture (EA) organization</strong> to define and enforce integration standards and reference architecture.</li>
	<li>Deep understanding of SOA, microservices, API management, security protocols (OAuth, SAML), and various data transformation techniques.</li>
	<li>Familiarity with Oracle ATP as a backend database for Oracle Fusion Cloud integrations</li>
</ul>
<h3><strong>Preferred Qualifications</strong></h3>
<ul>
	<li>Certification in Oracle Integration Cloud or other major middleware platforms (e.g., MuleSoft, Boomi).</li>
	<li>Experience architecting integrations for a migration from <strong>NetSuite</strong> to <strong>Oracle Fusion</strong>.</li>
	<li>Familiarity with the security and governance aspects of Oracle Cloud Infrastructure (OCI).</li>
</ul>
