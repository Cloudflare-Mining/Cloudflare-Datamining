<p><strong>About the Role:</strong></p>
<p>The Senior OIC and OPA Developer is the definitive technical expert responsible for developing, governing, and standardizing all data movement and orchestration involving the <strong>Oracle Fusion ERP Cloud</strong> environment. This role designs the most complex and critical integration patterns using <strong>Oracle Integration Cloud and Oracle Process Automation </strong>ensuring technical solutions adhere to Enterprise Architecture and security standards.</p>
<p><strong>Key Responsibilities</strong></p>
<ul>
	<li><strong>Complex Integration Design:</strong> Architect and develop the most challenging, cross-functional, and real-time integration patterns for the Oracle Fusion program using <strong>Oracle Integration Cloud.</strong></li>
	<li><strong>Architecture &amp; Governance:</strong> Act as the technical gatekeeper for all Oracle integration and process automation design, ensuring strict adherence to enterprise standards. Define the long-term technical roadmap for the OIC/OPA platform.</li>
	<li><strong>Technical Development:</strong> Personally drive the design and development of technical solutions (Interfaces, Conversions, Orchestrations and Workflows) within OIC, utilizing REST, SOAP, ERP, FTP, Other Application-specific Standard and Database adapters. Leverage Oracle AI Data (a.k.a Oracle ATP Database) for data staging and integration. Build BIP reports and ATP objects as needed to meet the requirements.</li>
	<li><strong>Security &amp; Performance:</strong> Define and validate the technical security protocols for all endpoints and interfaces (e.g., OAuth, API Gateway configuration), ensuring data transmission is compliant and performant.</li>
	<li><strong>Technical Mentorship:</strong> Serve as the top integration expert, providing technical leadership and direction to developers building integrations.</li>
</ul>
<p><strong>Required Qualifications</strong></p>
<ul>
	<li>10+ years of direct experience architecting, designing, and governing enterprise-level integrations for major ERP/SCM/GTM systems.</li>
	<li><strong>Expert-level, hands-on knowledge of Oracle Integration Cloud </strong>including strong familiarity with its various adapters, mapping, and orchestration capabilities.</li>
	<li>Proven experience working directly with a central <strong>Enterprise Architecture organization</strong> to define and enforce integration standards.</li>
	<li>Deep understanding of SOA, microservices, API management, and security protocols.</li>
</ul>
<p><strong>Preferred Qualifications</strong></p>
<ul>
	<li>Experience in an organization transitioning to Oracle Cloud Ecosystem.</li>
	<li>Hands-on experience with other integration tools like GoLang, Boomi, etc.</li>
	<li>Experience setting up/using CI/CD tools like FlexDeploy</li>
	<li>General awareness of built-in adapter based integrations in various 3rd party softwares.</li>
	<li>Exposure to the Data Center Infrastructure industry</li>
	<li>Relevant professional product/functional certifications (e.g., OIC/OPA, Database)</li>
	<li>Working knowledge of AI Agent Studio for Oracle Fusion Cloud Applications.</li>
</ul>
