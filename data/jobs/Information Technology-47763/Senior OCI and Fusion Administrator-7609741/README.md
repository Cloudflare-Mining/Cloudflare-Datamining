<p><strong>About the Role:</strong></p>
<p>This role is responsible for the technical administration, environment management, and ongoing platform integrity of the Oracle Fusion ERP Cloud environment, operating as a pure technical administrator for Oracle Fusion Applications and the underlying Cloud Infrastructure. The Administrator is a hands-on expert focused on system stability, security governance, and executing essential maintenance and support tasks.</p>
<p><strong>Key Responsibilities</strong></p>
<ul>
	<li><strong>Environment Management &amp; Maintenance:</strong> Own the technical management of all Fusion environments, including executing scheduled <strong>environment refreshes</strong>, cloning instances, managing environment usage, and ensuring system configuration baselines.</li>
	<li><strong>Cloud Update Execution:</strong> Participate in the technical preparation and execution of Oracle’s mandatory quarterly cloud updates, including performing pre-update checks and technical smoke testing post-update.</li>
	<li><strong>Platform Stability &amp; Governance:</strong> Own the non-functional requirements for the Oracle Cloud environment, including security architecture, role design governance, and performance benchmarking. Enforce technical configuration control standards.</li>
	<li><strong>Security Administration:</strong> Provide security administration and support for the all <strong>Oracle Fusion Applications, PaaS and DBaaS</strong> platforms, focusing on security key management, monitoring dashboards, and assisting with artifact deployment.&nbsp;</li>
	<li><strong>Risk Management Cloud:</strong> Own the Oracle Fusion Risk Management Cloud service as the technical owner and provide support to the Compliance business teams</li>
	<li><strong>Technical Support &amp; Troubleshooting:</strong> Provide Level 2/3 technical support for environment-related issues, access problems, and deployment failures. Serve as an escalation point to conduct root cause analysis for platform-level incidents.</li>
</ul>
<p><strong>Required Qualifications</strong></p>
<ul>
	<li>5+ years focusing on the technical administration/support within an <strong>Oracle Fusion environment</strong>.</li>
	<li>Expert-level knowledge of managing Oracle Fusion Cloud environments, including environment refresh and cloning processes.</li>
	<li>Deep technical familiarity with <strong>Oracle Cloud Infrastructure</strong> administration and monitoring.</li>
	<li>Strong understanding of security architecture, Oracle Fusion Risk Management Cloud, role design governance, and performance management within a cloud ERP.</li>
</ul>
<p><strong>Preferred Qualifications</strong></p>
<ul>
	<li>Experience in an organization transitioning to Oracle Cloud Ecosystem.</li>
	<li>Hands-on experience with various OCI PaaS Toolsets.</li>
	<li>Exposure to the Data Center Infrastructure industry</li>
	<li>Relevant professional product/functional certifications (e.g., Oracle Cloud Infrastructure and Security certifications)</li>
	<li>Skilled in administering DevOps tools like Flexagon FlexDeploy and using Opal IGA tool</li>
</ul>
