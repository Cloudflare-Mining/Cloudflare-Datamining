<h3><strong>Available Locations: Austin, TX or San Francisco, CA</strong></h3>
<h3><strong>About the Role</strong></h3>
<p>AI inference is becoming core infrastructure. Every serious application will need access to many models, across many providers, with reliability, observability, security, cost control, and routing built in from the start. AI Gateway is Cloudflare’s bet that this layer should exist at the network edge: close to users, close to compute, and simple enough that a developer can adopt it with one API change.</p>
<p>We are looking for a Senior Systems Engineer to help build that layer. This is a deeply hands-on individual contributor role for someone who can move from a half-formed product idea to a working, production-ready system quickly. You will work across the stack — from distributed systems and high-throughput APIs to developer-facing workflows, dashboards, SDKs, and docs — helping define how developers build and operate AI applications on Cloudflare.</p>
<p>You should be excited by the current pace of the AI ecosystem. Provider APIs change weekly, new models appear constantly, and developer expectations are rising just as quickly. AI Gateway needs engineers who can build durable infrastructure in that environment without slowing down.</p>
<p>We also believe AI should change how we build software, not just what we build. We dogfood AI Gateway, Workers AI, and Cloudflare’s broader agent stack heavily. You should be comfortable using AI-native tools, agents, harnesses, and automation to improve your own engineering loop — and interested in building the internal tools and context that help a small team ship like a much larger one.</p>
<h3><strong>Responsibilities</strong></h3>
<ul>
	<li>Build core AI Gateway systems that handle high-volume inference traffic across providers, models, and Cloudflare’s global network.</li>
	<li>Design and implement APIs, routing primitives, observability features, controls, and developer workflows that make AI Gateway reliable and easy to adopt.</li>
	<li>Work across product and infrastructure boundaries with Workers AI, Agents, and the broader Developer Platform organization to make Cloudflare’s AI platform feel like one coherent product.</li>
	<li>Turn ambiguous product and technical ideas into prototypes, design docs, shipped features, and measurable production improvements.</li>
	<li>Improve reliability, performance, cost efficiency, and debuggability for latency-sensitive AI workloads.</li>
	<li>Build the software — and the software that builds the software: agents, tests, harnesses, evaluation loops, operational tools, and internal context that help the team move faster with confidence.</li>
	<li>Sweat the developer experience. Error messages, SDKs, dashboard flows, docs, examples, and API details all matter.</li>
	<li>Mentor other engineers through design review, code review, debugging, and technical leadership without needing people-management authority.</li>
	<li>Bring strong opinions, clear judgment, and a willingness to revise your view when data, users, or production reality prove otherwise.</li>
</ul>
