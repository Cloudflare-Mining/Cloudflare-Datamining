####  A joint path to lower cost, faster delivery, and stronger resilience 

Cloudflare and Oracle work together to simplify multicloud operations while keeping performance and security consistent. Joint customers can use Cloudflare with OCI Object Storage to reduce transfer costs, route traffic over trusted network interconnections, and protect Internet-facing services with enterprise-grade controls. 

[  Read the OCI integration announcement ](https://www.cloudflare.com/press-releases/2025/cloudflare-integrates-services-with-oracle-cloud-infrastructure-to-help/) 

### Why use Cloudflare with Oracle

Adopt a shared architecture that helps teams move faster without sacrificing control.

**Reduce transfer cost pressure** 

Use Cloudflare with OCI Object Storage to avoid unnecessary egress costs and improve unit economics for global delivery. 

**Improve latency for users and services** 

Accelerate content and API traffic with Cloudflare's global edge and optimized network paths into Oracle infrastructure. 

**Strengthen edge-to-cloud security** 

Layer DDoS protection, bot defense, and Zero Trust controls across workloads that span Oracle and other cloud environments.
